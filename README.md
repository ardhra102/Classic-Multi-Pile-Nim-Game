# Classic-Multi-Pile-Nim-Game

A two-player console-based Classic Multi-Pile Nim Game built using Python. The project emphasizes logical thinking, game flow implementation, user input validation, and the practical application of core Python programming concepts.

# 📌 About the Game
The game starts with three piles containing different numbers of objects. Two players take turns choosing one pile and removing 1 to 3 objects from it. Players can remove objects from only one pile per turn. The game continues until all the piles become empty, and the player who removes the last remaining object wins.

# ✨ Features
Two-player console gameplay
Multiple object piles
Remove up to 3 objects per turn
Input validation for invalid pile names
Prevents removing more objects than available
Detects empty piles
Displays the remaining objects after every valid move
Announces the winner at the end of the game
